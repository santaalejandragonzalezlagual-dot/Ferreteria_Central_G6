## 📋 Descripción

**Ferretería Central** es una tienda/ferretería en línea donde los clientes pueden explorar un catálogo de herramientas, materiales y equipos de construcción. Los artículos comprados pueden ser **recogidos en tienda** o **enviados a domicilio**.

---

## 🛠️ Tecnologías

- **HTML5** — Estructura y contenido
- **CSS3** — Diseño y estilos visuales
- **JavaScript** — Interactividad y lógica del cliente
- **Supabase** — Base de datos y autenticación (próximamente)
- **GitHub Pages** — Despliegue
